# Code of Conduct

Statecraft is committed to providing a welcoming and respectful environment.
All contributors and community members are expected to follow this Code of
Conduct.

## Our standards

We follow the [Contributor Covenant](https://www.contributor-covenant.org/)
Code of Conduct (version 2.1). In short, it asks people to:

- Be considerate and respectful.
- Avoid harassment or discriminatory behavior.
- Be transparent and constructive in discussions.
- Report misconduct in a timely manner.

## Reporting

If you believe someone has violated this Code of Conduct, please report it to
the maintainers by opening an issue in the repository and describing the
incident.
