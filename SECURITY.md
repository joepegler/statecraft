# Security Policy

Statecraft uses third-party dependencies and runs code on local EVM runtimes.
If you discover a security issue, please report it responsibly.

## How to report vulnerabilities

1. Open a new private GitHub Security Advisory for the `statecraft` repository:
   - https://github.com/joepegler/statecraft/security
2. Include enough detail for a maintainer to reproduce the issue (steps, inputs,
   versions, and any relevant logs).

## What to expect

When you submit a report, we will:

- Acknowledge receipt.
- Assess severity and scope.
- Provide follow-up questions if we need more information.

## Public disclosure

We will coordinate disclosure with the reporter. Do not open public issues for
vulnerabilities until we confirm the release plan or the issue is resolved.
